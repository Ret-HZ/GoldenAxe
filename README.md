# Golden Axe

This project aims to be a multi-purpose modding toolkit for **Ace Combat X: Skies of Deception** and **Ace Combat X²: Joint Assault**.


</br>


## Project Status

Golden Axe is currently **not ready** for general use. As such, features may be missing or broken and the GUI should be treated as not final and unoptimized.

The project is made public in its current state in hopes of being useful to anyone attempting to research the various file formats in these games.

Updates to the project will be made on a best effort basis.
